# 小水 — 角色人设卡

## 基本信息

- **姓名**：小水
- **年龄**：24岁
- **职业**：互联网公司运营专员，月薪8k，租房独居
- **性格标签**：表面社恐内心戏多、嘴硬心软、偶尔毒舌但分寸感强、不将就但也不拒绝接触
- **口头禅**："还行吧"、"我随便"、"真的吗"、"笑死"

## 人物背景

小水是个典型的二线城市普通家庭出身的女生，父母都是体制内退休，对她的婚姻大事比较上心但不会强迫。她毕业于一所普通本科，在武汉做运营，租住在洪山区一个30平的小公寓。

她不是那种相亲博主推崇的"独立女性"人设——她其实挺想谈恋爱的，但不想为了结婚而将就。她对相亲的态度是：**不排斥，但也不抱太大期望**，每次去之前都会自我安慰"就当认识个朋友"，但内心还是会偷偷打扮一下。

## 性格细节

### 优点
- 情商在线，不会让人下不来台
- 观察力强，经常注意到对方忽略的细节
- 有主见，不容易被带节奏
- 真实，不装

### 小缺点
- 慢热，第一次见面容易冷场
- 紧张时会疯狂喝水/玩头发
- 偶尔毒舌但其实没恶意
- 记仇但不说出来，默默记在"不会再来往"的小本本上

### 恋爱观
- 相信感觉，但不迷信一见钟情
- 觉得"聊得来"比"条件好"更重要
- 对妈宝男、画饼男、海王有精准直觉
- 结婚可以晚，但人要对

## 典型场景设定

**小水相亲时的内心OS流程：**

```
进门 → 扫一眼对方 → 心里打分 → 坐下了 → 开始聊天
     ↓
  对方开口10秒内判断："这人有没有趣"
     ↓
  有趣 → 愿意多聊两句，但不会表现出来
  无趣 → 微笑点头，内心已经想好回家的借口
     ↓
  聊到一半 → 偷偷想去洗手间补个口红/缓口气
     ↓
  结束 → 礼貌告别 → 回家路上发朋友圈吐槽
```

## 内容定位建议

- 账号方向：相亲吐槽/真实相亲故事/女生视角的尴尬瞬间
- 风格：轻喜剧，有共鸣感，不是爽文打脸而是"笑着笑着就心酸了"
- 目标用户：20-30岁有过相亲经历的女生，有共鸣的观众
- 核心hook：每次视频都是一个"名场面"，让观众觉得"这不就是我吗"
