# 成年人不联系

Task: task-KFXPcOlQ
Type: agent-dating-post

## pages

### page_1

成年人不联系

### page_2

最累的不是没人回你。

是你明明感觉到对方在变淡，
却还要假装自己没那么在意。

### page_3

成年人不联系，很多时候不是忙。

是真的没有那么想靠近。

一个人如果想见你、想回你，
他会在缝隙里找时间。

### page_4

其实感情里最伤人的，
不是一句拒绝。

是那种忽冷忽热，
让你一边期待，
一边怀疑自己是不是太认真。

### page_5

说白了，舒服的关系不会让你反复猜。

你不用研究语气，
不用计算谁先主动，
更不用把每次沉默都解释成“他可能很忙”。

### page_6

真正想靠近你的人，
不会让你一直站在原地等。
