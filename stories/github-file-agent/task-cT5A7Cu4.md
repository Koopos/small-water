标题：怎么聊天

## page_1
怎么聊天，才能不显得太用力？

## page_2
很多成年人相亲，最累的不是没话题。

是你明明已经很认真了，对方却只回几个字。

你怕太主动显得掉价，怕太冷淡又错过机会。

聊着聊着，像在被观察，也像在接受一场没有说明书的面试。

## page_3
其实聊天这件事，真的不是谁话多谁就赢。

很多人一紧张，就开始拼命找话题、抛问题、证明自己有趣。

但成年人最真实的是：

如果对方没有接住你的情绪，你说再多，也只是一个人在撑场。

## page_4
男生一般会觉得：我主动了，你怎么还这么冷？

女生其实更在意的是：你是不是只想推进关系，却没有真的理解我？

说白了，聊天不是查户口，也不是表演幽默。

是你来我往里，能不能让对方觉得：跟你说话不用防着，也不用装着。

## page_5
慢慢你会发现，舒服的聊天都有一个共同点：

不是秒回，不是甜言蜜语，也不是每天从早聊到晚。

而是对方会顺着你的话往下走，愿意解释，愿意分享，也愿意让你靠近一点。

没有回应的热情，最后都会变成自我怀疑。

## page_6
真正合适的人，不会让你每句话都像在考试。
