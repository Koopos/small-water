# 相亲怎么聊不尬

## page_1

相亲怎么聊不尬

## page_2

很多人相亲最怕的，不是没话说。

是两个人坐在一起，明明都想表现得自然一点，却每句话都像在找答案。

你问一句，他答一句；他抛个问题，你又赶紧接住。

表面上在聊天，心里其实一直在想：我这样说，会不会很尴尬？

## page_3

其实相亲聊天，真的不用一开始就聊得很精彩。

很多人越想“不冷场”，越容易把聊天变成面试。

工作、收入、房子、家庭、计划……

这些当然重要，但如果一上来全是信息交换，两个人很难产生想继续了解的感觉。

## page_4

成年人最舒服的聊天，是有一点松弛感。

不是你必须句句有趣，也不是对方必须秒回。

而是你说一件小事，对方愿意顺着聊；
你开一个玩笑，对方不会立刻冷掉；
你停顿一下，也不用急着证明自己。

说白了，不尬的关键不是话题多，而是彼此都愿意接住。

## page_5

如果你不知道聊什么，可以少一点“查户口”，多一点“感受”。

别只问：你平时忙吗？
可以说：感觉你应该是那种做事挺稳的人。

别只问：你喜欢什么？
可以说：我发现你聊到这个的时候，整个人会放松一点。

聊天一旦有了观察和回应，就不会只剩干巴巴的问题。

## page_6

真正不尬的聊天，是两个人都不用太用力。
