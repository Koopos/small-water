# 相亲怎么推进关系

## Page 1
相亲怎么推进关系

## Page 2
相亲最怕的，不是对方条件不好，而是：

你感觉对了，但不知道该不该推进。

两个人吃了几次饭，聊了几次天，氛围还行，但你不敢牵手，怕太快；你不敢表白，怕被拒。

然后就开始耗着，谁也不主动，谁也不拒绝。

最后要么不了了之，要么变成了互相试探的博弈。

## Page 3
其实推进关系这件事，跟主动不主动关系不大。

真正的问题是：你们之间有没有形成「情绪共振」。

什么叫情绪共振？

就是他说的每句话，你愿意接；你开的玩笑，他愿意笑；你们聊天的时候，不是他在问你在答，而是两个人都在往同一个方向走。

如果没有这个，哪怕你天天主动发消息，也只是一个人在演戏。

## Page 4
很多人会陷入一个误区：以为推进关系 = 升级行为。

牵手、表白、确立关系……

但真正的推进，是从「相处模式」开始的。

比如：

他会不会在你面前暴露自己的脆弱？
你们有没有过一次，真正意义上的深入聊天？
聊天的时候，他是只在讲自己，还是会问你「你怎么看」？

这些细节，比牵手顺序重要一百倍。

## Page 5
所以，相亲里最舒服的推进方式，不是急着确定关系，而是：

先让两个人相处得「像自己」。

不用刻意找话题，不用假装很积极，不用每句话都斟酌。

而是你们都能在这个人面前，松口气。

推进关系的信号，不是他说了什么，而是你们之间的「空气」变了。

变得轻松了，变得有期待了，变得不害怕失去对方了。

## Page 6
真正对的人，不需要你用力推进。
